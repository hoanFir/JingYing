

### 会议的作用

1. 长期的会议缺失会减弱组织向心力

2. 是群体沟通、集思广益的渠道

3. 是信息交流的方式，具有面广、直接和高效的优势


### 会议的组织包括四个阶段

1. 会议策划

对会议活动的总体战略和进程进行前瞻性规划与设计的活动。

2. [会议筹备](https://github.com/hoanFir/JingYing/blob/master/%E4%BC%9A%E8%AE%AE%E7%BB%84%E7%BB%87%E4%B9%8B%E4%BC%9A%E8%AE%AE%E7%AD%B9%E5%A4%87.md)

对会议具体工作的事先准备与安排。

3. 会议实施

4. 会议结束


### 会议策划

1. 前期工作

1.1 确定会议策划者，一般是精干有丰富经验的人

1.2 收集信息，充分了解会议的目的与性质

1.3 ...


2. 策划过程中应遵循的四大原则

2.1 目的性原则

```

目的性是会议的重要特征，策划者要明确并围绕某一会议目的，如为了传递有关信息，或为了打造品牌形象，或推动销售额。


```

2.2 利益主导原则

```

策划者要秉持共赢理念，明确会议所追求的利益以及成本，合理配置资源。

```

2.3 整体规划原则

```

策划者应该具有全局观，做出周到的统筹规划。

```


2.4 可操作性原则

```

规划和设计应该结合客观情况，以及组织实际需要和能力才进行。

```
